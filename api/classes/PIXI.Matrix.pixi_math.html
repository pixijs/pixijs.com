Class: PIXI.Matrix

@pixi/math.Matrix

The PixiJS Matrix as a class makes it a lot faster. Here is a representation of it:

| a | c | tx|
| b | d | ty|
| 0 | 0 | 1 |

Constructor:

new PIXI.Matrix(a: number, b: number, c: number, d: number, tx: number, ty: number)

Parameters:

NameTypeDefaultDescription
anumber1x scale
bnumber0y skew
cnumber0x skew
dnumber1y scale
txnumber0x translation
tynumber0y translation
